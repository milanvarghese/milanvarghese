## 🚀 Featured Projects

- **[GapScope — Engineering Knowledge-Gap Analyzer](https://github.com/milanvarghese/GapScope-Engineering-Knowledge-Gap-Analyzer)** — A deployed LLM-powered career-gap analyzer (Next.js + Claude). Mines evidence from public GitHub repos, grounds it with a structured LLM pass, and returns a trajectory-aware learning path, project gaps, and a positioning read. Built with hallucination guardrails — HTTP-verified resource links and validated graph-shaped output.
- **[Political Bias Transfer in LLM-Generated Training Data](https://github.com/milanvarghese/Political-Bias-Transfer-in-LLM-Generated-Training-Data)** — A multi-axis audit of whether an LLM's political lean transfers into classifiers trained on its synthetic output. 189 fine-tuning runs across 7 generators and 3 architectures, with word-level mechanism analysis.
- **[Neuflo Learn Backend](https://github.com/NeufloSolutions/Neuflo-Learn-Backend-2023-24)** — A FastAPI + PostgreSQL backend powering an adaptive NEET exam-prep platform: closed-loop test generation, proficiency tracking, and per-chapter adaptive feedback.
